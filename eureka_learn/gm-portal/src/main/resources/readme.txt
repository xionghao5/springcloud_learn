在中国快速搭建一个小公司网站可以按照以下步骤进行：

1注册域名和选择托管服务：首先需要注册一个域名，并选择一个可靠的托管服务商。在中国常用的域名注册服务商包括万网、新网、DNSPod等，常用的托管服务商包括阿里云、腾讯云、华为云等。

2选择建站工具：在中国，常用的建站工具包括WordPress、Wix、Sitebuilder、ZCOOL等。选择一个适合你需求的建站工具，根据提供的模板和主题来构建网站。

3网站设计和排版：根据你的业务需求和网站目标来设计和排版你的网站。确保你的网站布局清晰、易于导航、色彩搭配合理。在网站中包含关键信息，如公司简介、产品和服务、联系方式等。

4网站内容填充：确保网站内容简洁明了，易于理解。包括公司简介、产品和服务、联系方式等。确保你的内容精炼，语言简单明了。

5优化网站：使用SEO技巧来优化网站，以便更容易在搜索引擎上被找到。使用关键词、描述标签、元标签等优化技巧，优化网站的速度和可访问性。

6测试和上线：在上线之前，确保测试你的网站。检查链接、图片、表单和其他功能是否都能正常工作。一旦你的网站准备好了，就可以将其上线并开始营销。

总之，通过注册域名和选择托管服务，选择一个适合你需求的建站工具，设计和排版你的网站，填充适当的内容，优化网站以便在搜索引擎上被找到，测试和上线你的网站，就能快速搭建一个小公司网站。
